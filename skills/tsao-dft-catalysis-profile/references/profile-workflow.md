# Profile Workflow

1. Create a profile record from `templates/profile.yaml`.
2. Enumerate independent chemical entities before conformers and methods.
3. Separate parent-molecule identity from conformer, spin state, coordination mode, and calculation level.
4. Build accepted minima before any gallery or ranking.
5. Select balanced reference states for monomer coordination, Ti/TEA association, and ligand exchange.
6. Run method-sensitivity checks where state orderings are close or transition-metal spin is central.
7. Use the project figure playbook only after accepted artifacts exist.
8. Export complete galleries to SI and retain source hashes, camera registry, isovalues, and comparison scales.
9. Phrase conclusions as supported, not supported, or unresolved within the declared model.
