# Ecosystem Integration

TsaoDFT uses Gaussian, Multiwfn, VMD/Tachyon, VASP, Quantum ESPRESSO, CP2K, pymatgen, ASE, AiiDA, atomate2, Psi4, DeepChem, RDKit, MDAnalysis, Cantera, RMG-Py, Pyomo, CatMAP, Snakemake and Nextflow as optional external ecosystems. The repository does not vendor them or override their licenses.
