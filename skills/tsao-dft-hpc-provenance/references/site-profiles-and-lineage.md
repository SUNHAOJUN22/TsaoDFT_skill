# Site Profiles, Resources and Restart Lineage

Site profiles record scheduler, executable/module version policy, scratch and resource limits without credentials. They are site-local evidence; public templates must not contain host secrets or licensed data.

Resource estimates report allocated CPU/GPU hours, not measured efficiency. Every restart records parent checkpoint hash and method fingerprint. An exact restart cannot change the scientific or numerical method; geometry or wavefunction reuse creates a distinct child run.
