# DFT Dataset and Model Cards

Every DFT label must retain parent structure, structure hash, method fingerprint, fidelity, units, acceptance status and uncertainty/source flags. The independent split unit is chosen before modelling. Conformers, spin states, charge states, repeated calculations and random seeds are not automatically independent samples.

A baseline model card records train-only preprocessing, group/scaffold/composition split, seeds, metrics, predictions, uncertainty/calibration and applicability domain. Feature importance remains model explanation, not chemical causality.
