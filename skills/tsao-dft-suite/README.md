# tsao-dft-suite

Repository-level DFT orchestrator. Use it for projects spanning structure preparation, molecular/periodic DFT, HPC, DFT+ML, kinetics, or optional chemistry profiles.

It produces a reviewed DFT task graph and validates cross-Skill handoffs. It does not run Gaussian, VASP, QE, CP2K, Multiwfn, or schedulers by itself.
