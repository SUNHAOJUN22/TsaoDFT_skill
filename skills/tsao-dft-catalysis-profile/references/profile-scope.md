# Profile Scope and Exclusions

## In scope

- DCS/MCSOMe/DMOS and related organosilicon substituent comparisons.
- Si-O/Si-C electronic and conformational effects relevant to monomer/catalyst interaction.
- Ti/TEA association and coordination-state hypotheses.
- Ziegler-Natta or related polyolefin catalyst coordination, inhibition, poisoning, and selectivity questions.
- Closed-shell monomer and open-shell Ti-complex comparison with explicit state handling.

## Out of scope by default

- Generic drug discovery, protein-ligand modeling, periodic solids, battery materials, and unrelated photochemistry.
- Claims about industrial catalyst performance without a defensible bridge from the molecular model to experiment.
- ML/SHAP from a small set of correlated conformers or multiple calculations from the same parent structure.

## Activation rule

Activate only when the user or supplied project identifies an in-scope chemical system. Never infer activation from the repository name alone.
