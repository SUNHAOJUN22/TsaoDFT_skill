# Third-Party Software and Inspiration

This repository contains original workflow code and documentation under MIT. It does not distribute Gaussian, Multiwfn, VMD, Tachyon, DeepChem, MDAnalysis, commercial manuals, basis-set databases, or third-party Skill repositories.

The architecture was informed by public ideas from:

- `thyheal/gaussian-workflow-copilot` — auditable Gaussian planning, preflight, diagnosis, and validation.
- `deepchem/deepchem` — molecular machine learning and leakage-aware splitting.
- `MDAnalysis/mdanalysis` — real molecular-dynamics trajectory analysis.
- `assafelovic/gpt-researcher` — planner/executor research orchestration and source tracking.
- `K-Dense-AI/scientific-agent-skills` — modular scientific Skills and progressive disclosure.
- `Imbad0202/academic-research-skills` and `academic-research-skills-codex` — claim/citation integrity and research gates.
- `Orchestra-Research/AI-Research-SKILLs` — research orchestration and rigor-review concepts.
- `JCLiuGroup/AI-Computational-Chemist` — durable task DAGs, accepted-artifact reporting, and human approval breakpoints.
- `jinzhezenggroup/computational-chemistry-agent-skills` — modular computational-chemistry Skill patterns.
- `Yuan1z0825/nature-skills` — conclusion-led figure design and submission-oriented QA.

No upstream repository is bundled wholesale. Review each upstream license before copying or vendoring any file. Users must cite the actual scientific programs and methods used in their work.
