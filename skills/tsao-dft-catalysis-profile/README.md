# tsao-dft-catalysis-profile

Optional profile for DCS/MCSOMe/DMOS, Si-O/Si-C, Ti/TEA, Ziegler-Natta and polyolefin-catalysis coordination or poisoning questions. It requires `tsao-dft-researcher` and must not be loaded for unrelated chemistry.

## v0.4 depth

See `SKILL.md`, `manifest.yaml`, `scripts/`, `templates/`, and `tests/` for the deterministic DFT adapters and scientific gates introduced in v0.4.0-alpha.1.
