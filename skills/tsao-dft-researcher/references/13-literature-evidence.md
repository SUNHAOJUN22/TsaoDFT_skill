# Literature, method fingerprints and claim evidence

## Research loop

```text
question decomposition
→ primary-source search
→ method/claim extraction with locators
→ evidence map
→ calculation design
→ result comparison
→ claim audit
```

Prefer official program documentation for executable syntax and peer-reviewed
primary sources for scientific methods and benchmarks. Record version/date for
software rules that may change.

## Method fingerprint from a paper

Capture molecular model, charge/spin, conformer/state scope, code/version,
functional/Hamiltonian, basis/ECP, dispersion, solvation, grid/SCF/optimization,
frequency/thermal treatment, standard state and post-processing. Mark missing
fields as reconstructed assumptions; do not silently fill them.

## Claim-evidence matrix

For each manuscript or report claim, record:

- claim text and strength;
- required observable;
- supporting artifact IDs;
- source locators or calculation paths;
- validation and acceptance status;
- alternative explanations and limitations.

No fabricated DOI, author, page, numerical value or program output is permitted.
When a source supports only a weaker statement, weaken the claim rather than
stretch the citation.
